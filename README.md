# create-backend
A CLI built in Node.js, to automate the process of creating a rest api / sockets backend basics

¡¡¡ DON'T INSTALL !!!

Use with npx anywhere (installation free)
> npx create-backend

Then you just answer less than 5 prompts and boom! you got a backend.
